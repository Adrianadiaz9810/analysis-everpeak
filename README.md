# Análisis de Comportamiento de Clientes ConnectaTel– Sprint 7

Este repositorio contiene el análisis realizado durante el Sprint 7 del Análisis de Comportamiento de Clientes ConnectaTel.

Los datasets /datasets/plans.csv, /datasets/users_latam.csv, /datasets/usage.csv incluye 4,000 datos de usuarios con valores faltantes, sentinels, outliers y problemas de calidad diseñados para simular datos reales del Análisis de Comportamiento de Clientes ConnectaTel

## 📘 Cómo reproducir el análisis

1. Clona o descarga este repositorio.
2. Coloca los archivos plans.csv, users_latam.csv y usage.csv 
   en la misma carpeta que el notebook.
3. Ejecuta todas las celdas en orden desde la primera.
4. Los gráficos y resultados se generarán automáticamente.

## 🧠 Objetivo del análisis

Analizar el comportamiento de uso de clientes de ConnectaTel, empresa de telecomunicaciones con operaciones en México y Colombia, con el fin de:

Identificar patrones de uso de llamadas y mensajes.
Detectar comportamientos atípicos o valores inusuales.
Segmentar clientes según edad y nivel de consumo.
Proponer recomendaciones comerciales para optimizar la oferta de planes.
